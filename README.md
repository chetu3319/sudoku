# sudoku
P5 js Sudoku

Controls :

Mouse : 
- click on a cell to select it
- click on your selected cell to incremente its value

Keyboard :
- Use arrows to move your selection
- Press a number key to affect value
- Press 0 to clear cell value
- Press any key to show/hide possible numbers in cell
